Крок 2

Йдемо на trinket.io/turtle

В адресному рядку (зверху) набери літерами латиниці: trinket.io/turtle — і натисни Enter. Дуже важливо: саме з /turtle в кінці.

Адресний рядок повинен виглядати так:
🔒 trinket.io/turtle
→ натисни Enter, щоб перейти
Уважно з адресою Якщо набрати просто trinket.io без /turtle — потрапиш на головну, де є кнопка «Python». Вона веде в інший режим (Python 3), де наш код працювати не буде. Тому одразу: trinket.io/turtle.
Що тут робить літера «io»? Це домен, як .ua чи .com. Багато технологічних сайтів обирають його, бо звучить як «I/O» (input/output, ввід/вивід).
Крок 3

Сторінка одразу відкрила редактор

trinket.io/turtle не показує жодних меню — кидає одразу всередину. На екрані вже є приклад коду від Trinket: він малює червоно-жовту "зірку". Не лякайся незрозумілих рядків — за хвилину ми все це приберемо.

main.py
# Default starter code from turtle import * color('red', 'yellow') begin_fill() while True: forward(200) left(170) if abs(pos()) < 1: break end_fill() done()
область результату поки порожньо —
▶ Run ще не натискали
Не запускай цей приклад Він написаний для старішої версії turtle і на trinket.io/turtle малює не зірку, а тонесеньку чорну рисочку. Не лякайся — це не твоя помилка. Ми зараз все одно видалимо цей код.
Червоний ⚠️ біля «main.py» — нормально Цей значок означає просто «файл змінений», а не помилку. З'являється і зникає сам по собі. Не звертай на нього уваги.

main.py
# Default starter code from turtle import * color('red', 'yellow') begin_fill() while True: forward(200) left(170) if abs(pos()) < 1: break end_fill() done()

Крок 5

Стираємо приклад, пишемо своє

Виділи весь код у лівому вікні (Ctrl + A) та видали (Delete). Тепер пиши ось це:

main.py
import turtle import random (поки що все)
поки що порожньо —
натиснемо ▶ Run на наступному кроці
Що це за import?

Команда import підключає бібліотеку — готовий набір функцій, які хтось вже написав для нас. turtle вміє малювати. random вміє вибирати випадкові речі. Без них наша програма буде сама не своя.

Крок 6

Кличемо черепаху на сцену

Додаємо ще 2 рядки. Перший створює черепаху-художницю на ім'я t. Другий — каже їй малювати максимально швидко.

main.py
import turtle import random t = turtle.Turtle() t.speed(0)
Натисни ▶ Run
Що повинно з'явитися Маленький чорний трикутничок (стрілочка) посередині правого вікна. Це і є наша черепаха. Поки вона дивиться вправо.
Чому speed(0)? Дивно, але у turtle 0 — це максимальна швидкість, а не "стоп". 1 — найповільніша, 10 — швидка, 0 — миттєво. Логіка інопланетна, але звикнемо.
Крок 7

Малюємо перший квадрат

Додаємо цикл: 4 рази "пройти вперед на 100, повернути направо на 90°". Виходить квадрат.

main.py
import turtle import random t = turtle.Turtle() t.speed(0) for _ in range(4): t.forward(100) t.right(90)
Після ▶ Run
Розбираємо for _ in range(4):

Це означає: "виконай те, що відступлено праворуч, рівно 4 рази". Підкреслення _ — це змінна, яка нам не потрібна (просто лічильник). Двокрапка та відступ — обов'язкові!

Крок 8

Розфарбовуємо

Перед циклом скажемо черепасі, якого кольору лінія. А фон зробимо чорним — на ньому кольори будуть яскраві.

main.py
import turtle import random t = turtle.Turtle() t.speed(0) screen = turtle.Screen() screen.bgcolor("black") t.color("red") for _ in range(4): t.forward(100) t.right(90)
Після ▶ Run
Замість «red» спробуй "yellow", "cyan", "magenta", "lime", "orange". Усі назви — англійською, у лапках. Лапки можна одинарні або подвійні — без різниці.
Крок 9

Готуємо палітру

Створимо список кольорів. Кожен квадрат буде вибирати з нього випадковий колір.

main.py
import turtle import random t = turtle.Turtle() t.speed(0) screen = turtle.Screen() screen.bgcolor("black") colors = ["red", "yellow", "blue", "green", "orange", "cyan"] t.color(random.choice(colors)) for _ in range(4): t.forward(100) t.right(90)
Запусти кілька разів!
Експеримент Натисни ▶ Run 3–4 рази підряд. Колір квадрата щоразу різний! Це робота random.choice(colors) — обирає випадковий елемент зі списку.
Крок 10

36 квадратів — і виходить розетка

Загортаємо весь блок (вибір кольору + малювання квадрата) у ще один цикл, на 36 повторень. Між квадратами повертаємо черепаху на 10° — так усі вони стануть під різним кутом і разом утворять зірку.

main.py · фінальний код
import turtle import random t = turtle.Turtle() t.speed(0) screen = turtle.Screen() screen.bgcolor("black") colors = ["red", "yellow", "blue", "green", "orange", "cyan"] for i in range(36): t.color(random.choice(colors)) for _ in range(4): t.forward(100) t.right(90) t.right(10)
Розетка! 🎉
Обережно з відступами! Малий цикл (4 рази) тепер всередині великого (36 разів) — значить його рядки мають подвійний відступ (8 пробілів). Python суворо стежить за відступами.
Завдання 1

Експерименти з параметрами

Спробуй змінити одне число в коді й подивись, що буде. Без страху — нічого не зламається.

A

Заміни 36 на 72

В рядку range(36) постав 72. Квадратів стане вдвічі більше — розетка буде щільніша.

B

Заміни кут 10 на 5

Останній рядок t.right(10) → t.right(5). Квадрати рідше повертатимуться — вийде "вузенька" квітка.

C

Заміни 100 на 50

В t.forward(100) постав 50. Квадрати стануть маленькі — мікророзетка!

Порада Перед кожним експериментом — натисни ▶ Run, щоб побачити результат. Не подобається? Поверни число назад або вигадай нове.
Завдання 2

Своя фігура замість квадрата

Квадрат — це 4 сторони, кут повороту 90°. А якщо взяти іншу фігуру? Правило: кут = 360 ÷ кількість сторін.

△

Трикутник

range(3):
  forward(100)
  right(120)

⬠

П'ятикутник

range(5):
  forward(80)
  right(72)

⬡

Шестикутник

range(6):
  forward(70)
  right(60)

Шпаргалка по кутах Трикутник: 360 / 3 = 120° · Квадрат: 360 / 4 = 90° · П'ятикутник: 360 / 5 = 72° · Шестикутник: 360 / 6 = 60° · Восьмикутник: 360 / 8 = 45°
Бонус для швидких

Спіраль замість розетки

Що буде, якщо в кожному циклі квадрат робити трохи більшим, ніж попередній? Перетвориться на закручену спіраль.

Зміни в коді
main.py
# ...решта коду без змін... for i in range(100): t.color(random.choice(colors)) t.forward(i * 3) t.right(91)

Тут i * 3 — це довжина кроку. Першого разу i = 0, потім i = 1, потім 2... і кожен крок довший. А right(91) замість 90 — це секрет, який і робить спіраль "закрученою".
